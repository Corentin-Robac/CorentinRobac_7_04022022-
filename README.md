# Groupomania - Project 7 Openclassrooms Web Dev

## Installation
```
git clone https://github.com/Corentin-Robac/CorentinRobac_7_04022022-.git Groupomania
cd front
npm install
cd ../back
npm install
```

## Load Front
```
npm run serve
```

## Load Back
```
nodemon server
```
